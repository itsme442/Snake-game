▎Snake Game

A classic Snake game implemented in HTML, CSS, and JavaScript. Navigate the snake to eat food, grow longer, and avoid collisions. This project showcases fundamental programming concepts and offers an engaging user experience.

▎Features:
- Responsive design for various devices
- Real-time score tracking
- Game over detection and restart functionality

▎Getting Started:
Clone this repository and open index.html in your browser to play!
